# Distributed Salary Systems (DSS)

Blockchain-based infrastructure for contractor management and cross-border payouts.

## Overview

Distributed Salary Systems (DSS) is a hybrid on-chain/off-chain payment infrastructure designed to enable fast, transparent and cost-efficient contractor payouts worldwide.

DSS leverages BNB Smart Chain and tokenized incentives to reduce operational friction in global payroll and contractor management.
